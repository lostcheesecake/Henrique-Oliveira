I thrive in the face of a challenge because I love the satisfaction that comes from overcoming it.

What Are Those!?!

An app using Ruby on Rails as the backend, and React.js as the frontend, that allows a signed in user to upload a sneaker of their choice which includes the brand, color, and style. They are able to click on the sneaker they uploaded and either update it if they would like, or delete it. The user is also able to see the sneakers that other users have uploaded.

Bucket List

Group project done with Express as the backend and Vanilla.js as the frontend in which you are capable of adding something in life you want to accomplish to a list that contains all other life events you want to accomplish. The user can either update that event, delete it, or decide to complete it.

Video Game Review

FullStack project done using Ruby on Rails for backend, and Vanilla.js as frontend where a user can create a review for either a video game title, or a game console of their choice and upload it to a list of reviews. The user is able to update that review or delete it, along with being able to see reviews done by other users.
